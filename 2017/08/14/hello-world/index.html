#archLinux安装教程

# lsblk（个人建议分一个区 sda1）

# mkfs.ext4 /dev/sda1 //格式化

# mount /dev/sda1 /mnt  //挂载

# cd /etc/pacman.d //这里是源文件

# grep -A 1 '##.*China' mirrorlist|grep -v '\-\-'> mirrorlist2  //正则表达式保留中国源
# cat mirrorlist>>mirrorlist2
# mv mirrorlist2 mirrorlist

# pacman -Syy //刷新

# pacstrap -i /mnt base base-devel //开始安装系统

# genfstab -U /mnt >> /mnt/etc/fstab

# arch-chroot /mnt

# ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime //设置时间

# hwclock --systohc --utc   //设置为utc

# nano /etc/locale.gen
//填入以下内容，然后按ctrl+x按y回车保存退出
en_US.UTF-8 UTF-8
zh_CN.UTF-8 UTF-8

# locale-gen

# echo LANG=en_US.UTF-8 > /etc/locale.conf

# echo myhostname > /etc/hostname
//myhostname 可以自定义

# nano /etc/hosts
//在 ::1        localhost.localdomain    localhost 的下一行添加以下内容，myhostname 和上文的一致
127.0.1.1    myhostname.localdomain    myhostname

# mkinitcpio -p linux

# passwd root
设置root密码

# useradd -m -g users -s /bin/bash user
//添加用户，user 可自定义
# passwd user
设置用户密码，user与上文相同



# pacman -S efibootmgr grub os-prober (efi引导）
# grub-install --efi-directory=/boot --bootloader-id=grub(efi引导）

pacman -S os-prober    多系统用户一定要添加
#pacman -S grub-bios传统
#grub-install /dev/sda传统---如果报错添加--force参数就好了
grub-install --force /dev/sda


# grub-mkconfig -o /boot/grub/grub.cfg

# systemctl enable dhcpcd

# exit

# umount -R /mnt

# reboot



修改开机如把win设置为默认


    默认情况下Windows 7对应的文件名是30_os-prober，第一个linux系统对应的是10-linux，00是header，05是debian-theme，可见10是第一个启动项，只需要更改Windows 7的文件名(06-09均可)即可改变启动顺序，经我测试，成功地修改了启动顺序。

    具体做法：
    sudo mv /etc/grub.d/30_os-prober /etc/grub.d/08_os-prober
    sudo update-grub

安装yaourt 
Yaourt是archlinux方便使用的关键部件之一，但没有被整合到系统安装中的工具。建议在装完系统重启之后，更新完pacman和基本系统之后，就安装这个工具。 
最简单安装Yaourt的方式是添加Yaourt源至您的 /etc/pacman.conf，在文件最后加入:
[archlinuxcn]
#The Chinese Arch Linux communities packages.
SigLevel = Optional TrustAll
Server = https://mirrors.ustc.edu.cn/archlinuxcn/$arch
  ● 1
  ● 2
  ● 3
  ● 4
然后，用命令
# pacman -Syy
  ● 1
清理一下pacman数据库缓存，应该能看到archlinuxcn的字样。 
最后用
# pacman -S yaourt
  ● 1
安装Yaourt部件。
